# 00-static-html-css-js

A no-framework baseline template showcasing:
- Semantic HTML + accessibility
- Responsive layout + design tokens
- ES module JavaScript + simple state/render loop

## Run
Open `node server.js` and navigate to `http://localhost:3000`

## What to look for
- Landmarks: header/nav/main/footer
- Skip link + focus-visible styles
- CSS tokens + responsive grid
- JS modules: state + DOM helpers + render function
